#   线性方程组的解

## 相融  有解

线性方程组有解.

## 不相容  无解

线性方程组无解.

## A x = b 解的判定

- 无解 $\iff R(A) < R(A, b)$
- 有惟一解 $\iff R(A) = R(A, b) = n$
- 有无限多解 $\iff R(A) = R(A, b) < n$

## n 元齐次线性方程组 A x = 0 有非零解

$$\iff R(A) < n$$

## A x = b 有解

$$\iff R(A) = R(A, b)$$

## 矩阵方程 A X = B 有解

$$\iff R(A) = R(A, B)$$

若 $AB = C$, 则

$$R(C) \leqslant \min\{R(A), R(B)\}$$
